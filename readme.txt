= Credits =

Foram utilizados os seguintes scripts/serviços de terceiros:

* [Autopreenchimento de endereço em formulários Por FabbricaWeb ](https://br.wordpress.org/plugins/cf7-cep-autofill/).
* [CNPJ.ws](https://www.cnpj.ws/docs/intro).

== Installation ==

1. Faça o upload dos arquivos do plugin no diretório `/wp-content/plugins/cnpj-autofill`
2. Ative o plugin.
